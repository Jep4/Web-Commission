 * 설치 방법

  1. 압축을 푸신 후 wclapcfg.php 파일을 메모장으로 열어 적절히 수정합니다.

  2. readme.txt 파일을 제외한 폴더 안의 모든 파일을 계정에 올립니다.

  3. 그 후, "반드시" wclaplog.cgi 파일의 퍼미션을 606 혹은 666 이상으로 합니다.

  4. 웹박수를 실행할 시 - smwclap.php
     보내진 박수 및 메시지 확인 - wclapview.php

     에서 확인합니다. 잘 작동되면 설치 끝.


 * 환경 설정

  wclapcfg.php 파일에서 할 수 있는 환경설정에 대한 설명입니다.

  1. 랜덤 메시지 : 
       $ranmsg[] = '내용'; 형식으로 답례 메시지를 추가합니다.
       개수 제한은 없으므로 원하시는 만큼 추가하실 수 있습니다.
       단 그림 파일을 넣으실 때는 경로를 꼭 확인해 주세요.
       또한 형식은 작은 따옴표(')로 시작해서 작은 따옴표로 닫으므로 주의하세요.

  2. 패스워드 :
       wclapview.php 를 열람할 때 아무나 접근할 수 없게 하고 싶으시면
       패스워드를 지정해 주시면 됩니다.
       누구나 열람할 수 있도록 하고 싶으면 $adminpw = ''; 로 바꿔주세요.

  3. 메시지 개수 한도 :
       장난이나 테러를 방지하기 위해서 한 번에 보낼 수 있는 메시지 개수를
       제한할 수 있습니다.
       원하지 않으신다면 $msg_limit = 0; 로 바꿔주세요.

  4. 암호화 salt값 :
       salt값을 다르게 지정해 주시면 보다 보안을 강화할 수 있습니다.
       아무 문자열(영문 또는 숫자)이나 2자리로 적당히 넣어주시면 됩니다.

  5. HTML 출력 부분

       - $head    : HTML에서 공통적으로 상위에 표시되는 부분입니다.
                    내용은 <<<HEAD 와 HEAD; 사이에 입력해 주세요.

       - $thanks  : 박수를 보냈을 때 출력되는 부분입니다.
                    랜덤 메시지와 시간 정보가 여기에 들어갑니다.
                    내용은 <<<THANKS_PAGE 와 THANKS_PAGE; 사이에 입력해 주세요.

                    (!--random_message--)는 랜덤 메시지를 치환합니다.

       - $confirm : 메시지 전송이 성공하면 보여주는 화면입니다. (3초)
                    전달된 메시지를 확인차 다시 보여줄 수도 있습니다.
                    내용은 <<<CONFIRM_PAGE 와 CONFIRM_PAGE; 사이에 입력해 주세요.

                    (!--sent_message--)는 전송된 메시지를 치환합니다.

       - $caution : 설정된 메시지 개수 한도를 초과했을 때 보여지는 화면입니다.
                    내용은 <<<CAUTION_PAGE 와 CAUTION_PAGE; 사이에 입력해 주세요.

                    (!--limit--)는 설정된 개수를 치환합니다.

       - $foot    : HTML에서 공통적으로 하단에 표시되는 부분입니다.

       - 그 밑의 부분은 특별히 수정할 필요가 없는 부분입니다.


 * FAQ

  1. 아무리 박수를 보내도 저장되지 않아요!

     A. 로그 파일(wclaplog.cgi)의 퍼미션(권한)이 알맞게 설정되었는지 확인합니다.

        그리고 웹호스팅 서비스 업체에 따라 퍼미션을 606 혹은 666으로 설정해도 저장
        되지 않는 경우가 있다고 합니다. 이럴 때는 777로 설정해 보세요.
        (제보해 주신 분께 진심으로 감사드립니다!)

  2. Parse error: parse error, unexpected T_STRING... 라는 메시지가 떠요!

     A. 환경설정 파일(wclapcfg.php)의 내용은, HTML 출력 부분을 제외하고는 모두
        $변수명 = '문자열'; 형식으로 이루어져 있습니다.
        작은 따옴표(')를 정확히 열고 닫았는지, 마지막에 세미콜론(;)을 넣었는지,
        (패스워드를 제외하고) 빈 부분이 있는지 확인합니다.

  3. 환경설정 파일에 오류가 있다는 메시지가 떠요!

     A. 2번 답변을 참고해 주세요.

  4. 메시지가 제대로 보내지지 않아요!

     A. 환경설정 파일을 열어 HTML 출력 부분, 특히 $thanks 를 다시 확인합니다.
        그 안에 <form action="smwclap.php" method="post"> 와 </form> 가
        반드시 포함되어 있어야 합니다.
